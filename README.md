# Data Analyst Internship – Task 1 (Excel Version)

## Objective
Clean and prepare a raw dataset (Mall Customer Segmentation Data) by handling missing values, duplicates, inconsistent formats, and incorrect data types.

## Tools Used
- Microsoft Excel

## Steps Performed
1. Identified missing values and filled them using average (for numbers) and “Unknown” (for text).
2. Removed duplicate records using the "Remove Duplicates" feature.
3. Standardized text fields (like gender and country names).
4. Converted date columns into a consistent format (dd-mm-yyyy).
5. Renamed column headers to lowercase and without spaces.
6. Checked and corrected data types (number, text, date).
7. Saved the cleaned dataset as Cleaned_Mall_Customers.xlsx.
